### [](#0qxhah)MVC代表的含义：
```
MVC：Model + View + Controller(数据模型，视图，控制器)
数据模型Model是和View视图之间进行传值交互数据的，Controller 是控制器，用来管理注解@Controller的类
```

## [](#0xevkk)Spring Boot优点缺点
```
优点
1.快速构建项目
2.对主流的开发框架无配置集成
3.项目可以独立运行，无需依赖外部servlet容器
4.提供运行时的应用监控
5.极大地提高开发、部署效率；
6.与云计算的天然集成
```
```
缺点
1.书少文档不多不够深入
```
